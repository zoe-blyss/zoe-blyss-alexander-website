The Project

Project for my Web + Screen class Fall 2019 in which we were tasked with creating a website that uses unconventional page linking. Each page has a different pattern to where the hidden page links are. A more artistic approach to web design, each page has a different theme which is based on humorous imagery and phrases taken out of science textbooks.
