The Project

Final project for my Web Development class Fall 2019. Assignment was to replicate the home-page of a website. First tasked with breaking down all of the individual components and creating a style guide using these components, then replicating the component of site on a separate page using the BEM method, and finally compose all of atoms and molecules together to create the main page.

Original homepage of site I replicated, “French Kiss Life,” for comparison: https://frenchkisslife.com/
