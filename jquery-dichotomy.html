The Project

Project for my Web + Screen class Fall 2019. Was tasked with creating a dichotomy between two ideas using jQuery for animation. Explored the dichotomy of time and fate though a fallen leaf (its fate during the transitional season of Fall) and through a reference to the movie Donnie Darko in which the main character is destined to die at a specific time in order to save the world.
