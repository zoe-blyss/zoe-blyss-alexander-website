About Zoe

Zoe Blyss Alexander is a web designer currently living in Minneapolis, Minnesota. She earned her Bachelor of Science in Entrepreneurial Studies at the Minneapolis College of Art and Design (yes, she did go to art school to get a Bachelor of Science in business). Zoe particularly enjoys combining front-end development tools with art to push the current definitions of what websites and art can be. In her spare time Zoe likes read, play point-and-click interactive games (while analyzing how she would create them herself), and create digital collages.

Portfolio

Breakthrough Family Therapy

Business website completed for a client starting her Marriage and Family Therapy practice. Created both website and business cards for client using color palette and imagery requested. Uses responsive design.

The French Kiss Life Homepage Case-Study

The Gods of Ancient Egypt

A project for my Web Development class Fall 2019. Tasked with creating a responsive website of any sort, so choose to create one about the Ancient Egyptian gods and goddesses. Used a grid layout.

p5.js Final

Final project for Coding Concepts class Fall 2019. Assignment was to create a “choose-your-own-adventure” story through linked pages using JavaScript library p5.js.

Labyrinth-to-Net

Project for my Web + Screen class Fall 2019 in which we were tasked with creating a website that uses unconventional page linking. Each page has a different pattern to where the hidden page links are. A more artistic approach to web design, each page has a different theme which is based on humorous imagery and phrases taken out of science textbooks. Non-responsive design.

Abstraction

Project for my Web + Screen class Fall 2019. Was tasked with creating “abstractions” of phenomena occurring in daily life. The overarching theme was to explore the different types of breaks I get to take during a school day, as well as contemplate how relaxing they are or are not. Non-responsive design.

jQuery Dichotomy

Project for my Web + Screen class Fall 2019. Was tasked with creating a dichotomy between two ideas using jQuery for animation. Explored the dichotomy of time and fate though a fallen leaf (its fate during the transitional season of Fall) and through a reference to the movie Donnie Darko in which the main character is destined to die at a specific time in order to save the world. Non-responsive design.
